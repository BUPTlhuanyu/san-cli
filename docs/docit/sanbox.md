# 在 Markdown 中执行 San 代码
在 Markdown 中是可以运行 San 代码的，即San 代码是直接可以被执行，并且显示结果。这个功能常用在 Component 的文档编写中，我们在写 Component 时，经常遇见的问题就是写完组件还需要写个「测试页面」来查看 Demo 效果，测试没问题了之后，还需要写组件文档，在 San docit 中，文档即Demo！
