# @baidu/xbox-native

> 手百端能力函数库，代码从 filter 和 versionMap 摘出来的！

## 安装

```bash
npm install @baidu/xbox-native --registry http://registry.npm.baidu-int.com
```

## 使用

```js
import toast from '@baidu/xbox-native/ui/openToast';
```
